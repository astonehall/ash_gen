# Roadmap

## Phase 1 - MVP Backend (In Progress)

- [x] FastAPI service with health and generation routes
- [x] Typed request/response schema
- [x] Config and environment handling
- [x] Stub generation service with deterministic output contracts
- [x] SDXL runtime integration (single model)
- [ ] Basic job status endpoint

## Phase 2 - Core Product

- [ ] Minimal cross-platform UI shell
- [ ] Image gallery and generation history
- [ ] Authentication and session management

## Phase 3 - Advanced Controls

- [ ] LoRA support
- [ ] ControlNet support
- [ ] Upscaling and fix workflows
- [ ] Multiple model management

## Engineering Rules

- Keep dependencies minimal and justified.
- Prioritize Linux Mint stability first.
- Ensure compatibility with Windows and macOS paths/process behavior.
- Keep docs close to implementation updates.
